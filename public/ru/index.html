Почему?

Windows - проприентарное, дорогущее и медленное говно состоящее из костылей из 1980x годов без нормальной безопасности и огромным количеством вирусов.
Microsoft не заботится о юзерах и пилит винду только ради заработка денег которые отдают им компании и слишком богатые юзеры.

Компьютер с Windows принадлежит не вам.
Компьютер с Windows принадлежит корпорации.

Телеметрия и слежка которую невозможно нормально отключить.
Обновления Windows которые фиг отключишь.
Прекращение поддержки старых Windows и затем постоянные попытки пересадить насильно юзеров на новые версии.
Огромное количество вирусов которые появляются из-за того что винда изначально сделана криво.
И костыли в виде антивирусов которые жрут кучу ресурсов компьютера и далеко не всегда справляются с угрозами.
Реестр который является устаревшим и уязвимым способ хранить конфиги системы.
Улиточная скорость работы и загрузки системы.
На этот допотопный кусок говна Microsoft из года в год натягивает красивый UI, при этом забивая на все остальное (привет Windows 11).

И это далеко не всё...

  • Проприентарная файловая система не чуствительная к регистру и которая нормально поддерживающаяся только в Windows.
  • Синие экраны смерти не дающие подробную информацию для эффективного решения проблем.
  • Костыльная звуковая система.
  • Никакой нормальной кастомизации как и GUI так и всего остального.
  • Никакой нормальной безопасности из-за которой требуются антивирусы которые жрут кучу ресурсов.
  • Никакого нормального стандарта пакетов и репозиториев из-за чего приходится лезть куда-то в интернет при этом стараясь не подхватить вирусни.
  • Никакого нормального терминала.
  • Старое WinAPI из-за которого невозможно нормально писать программы под винду.
  • Низкая производительность OpenGL.
  • Закрытость системы из-за которой приходится делать костыли типа анти-читов в виде драйверов.
  • Кучу непонятных процессов которые запущены в фоне не понятно зачем.
  • Костыли в виде планировщика задач который нормально не работает.
  • Новые обновления не тестируются нормально из-за чего постоянно возникают какие-то проблемы.
  • Отвратительные заплатки от уязвимостей типа Meltdown и Spectre.
  • Система поиска и исправления ошибок которая никогда не помогает, а иногда портит систему.
  • Никакой нормальной системы управления зависимостями для программ из-за чего программы начинают всё больше и больше весить.
  • Не понятные решения (например акселерация мыши и постоянно растущая в размере AppData) которые никто не исправляет.
  • Постоянное перетирание GRUB и других загрузчиков на загрузчик Windows.
  • Костыльная поддержка мониторов с большим разрешением.
  • Везде уязвимости как и в ядре так и в user-space.

Разоблачение заблуждений про Linux

На Linux мало игр

На самом деле на Linux достаточно нативных игр, а те которые не нативные можно попробовать запустить через Wine или Proton.

Linux сложный

Нет. Существует много дистрибутивов (Linux Mint, KDE Neon и другие) в которых для нормального использования не нужно использовать терминал. Также на самом деле терминальные команды очень простые и легко запоминаются.

Linux уродливый

Для Linux существует очень много окружений рабочего стола и у всех них разный дизайн. Из наиболее красивых - KDE Plasma, GNOME и Cinnamon.

На Linux мало программ

Для большинства программ которых нет для Linux есть альтернативы. Например вместо Photoshop можно использовать GIMP, PhotoGIMP, или Krita. Если альтернативу найди не удалось, можно попробовать запустить это программу через Wine.

Linux поддерживает не все устройства

На самом деле поделать с этим мало чего можно (разве что ждать или купить другое устройства). Но нужно помнить что это не проблема Linux а проблема производителя устройства.

Для Linux нет клиента Epic Games Store

Существует неофициальный клиент для Epic Games Store - Legendary. Он консольный, но для него есть GUI - Heroic. Также для большей функциональности вы можете для этого использовать Wine и Lutris. Установите Lutris, из него установите EGS, и готово. Учтите что это может работать менее стабильно.
